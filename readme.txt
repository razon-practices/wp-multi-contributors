== Description ==

Add multiple contributors to a post and display them on the frontend with their Gravatars.

== Installation ==
1. Upload the `wp-multi-contributors` folder to `wp-content/plugins/`.
2. Activate the plugin via the WordPress admin panel.
3. Navigate to "WP Multi Contributors" to configure.
